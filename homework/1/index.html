Homework #1 - Birthday Wish List

Create one webpage using HTML that is a wish list of things you would like to receieve for your birthday.


Requirements

  • Your code must at least contain the <html>, <head>, <title>, and <body> tags.
  • Your code must use the <ul> and <li> tags to create a list of at least 5 things for your wish list.
  • Each <li> tag must have either:
    • a description of the gift, or
    • a nested <a> tag with a link to the gift.
  • Your code must use at least 2 colors.

Submission

You will turn in your homework by submitting a pull request to our class github repository and adding your index.html file to the folder corresponding to your NetID under /homework/1/ (i.e. cli-2767.github.io/homework/1/jacobsom/).


Discussion

Come to class next week prepared to discuss your choices. Some things to think about: How could you make this website better? How could you add some sort of interaction?
